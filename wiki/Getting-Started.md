## 🛠️ Installation and Setup

Follow the TTT-stabilized sequence to initialize the engine:

#### 1. Clone the Manifold
```bash
git clone https://github.com/Nexus-Resonance-Codex/Phi-Infinity-Lattice-Compression.git
cd Phi-Infinity-Lattice-Compression
```

#### 2. Synchronize Dependencies
```bash
uv sync
```

#### 4. Build the Compression Core
```bash
cargo build --release
```

#### 5. Verify Lattice Resonance
```bash
cargo test
```
